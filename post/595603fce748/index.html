前言

最近因为学习AIGC的原因需要学习Python，因为用习惯了VSCode的各种快捷方式，因此做了一下python相关的配置，方便自己开发
此文主记录Python相关的软件、VSCode插件安装、Python文件编译等

软件准备

下载地址：

  • Python（选择自己需要的版本下载）：https://www.python.org/downloads/
  • VScode：https://code.visualstudio.com/

软件安装

VSCode

VSCode安装很简单，基本上都是直接“下一步”即可

  • 在“同意协议”之后，会进入更换安装目录的步骤，需要更换的直接更换即可

VSCode默认是英文版本，需要汉化的可安装以下插件（如何安装插件后面有说明）
插件：Chinese (Simplified) (简体中文) Language Pack for Visual Studio Code


Python

具体的安装的各个环境介绍可参考：http://t.csdnimg.cn/PB6hA

  1. 勾选 Add Python x.xx to PATH，可以省去自己配置环境变量的步骤
  2. 点击 Customized installation，进入下一步，此步骤注意下 py launcher 有没有勾选，没有则勾上即可
  3. 点击next，进入下一环节，修改 安装路径，换成自己方便找的位置
  4. 点击 install，等待安装完成即可

检查Python是否安装成功

检查方法如下：

  1. win + r，输入cmd，回车
  2. 输入python，回车，出现一下界面则表示成功
  3. 可以输入 exit() 退出当前模式，接下来的步骤还会使用~

安装第三方的包

使用Python自带的 pip 工具，安装 flake8、yapf

  • flake8会检查编写代码时的不规范的地方和语法错误
  • yapf是一个代码格式化工具，可以一键美化代码

安装过程：

  1. 在cmd中输入 pip list，检查需要安装的包是否已经安装，没有则进行下一步
  2. 分别在cmd中输入以下命令，安装flake8和yapf

VSCode 配置

Python相关插件安装

操作步骤：

  1. 打开VSCode，按下快捷键Ctrl+Shift+X，进入插件管理页面
  2. 在搜索框中输入以下插件名，点击对应的插件，安装（install）即可

需要安装的插件主要有以下3个：

  • *Python：提供对 Python 语言的支持
  • *Python Debugger：用于 Visual Studio Code 的 Python 调试器扩展
  • Python Snippets 3：Python3 代码包片段提示

创建项目

  1. 提前在创建一个用来放Python代码的文件夹
  2. 在VSCode的资源管理器中，点击“打开文件夹”，选择创建好的文件夹
  3. 可能会出现以下步骤，点击信任即可
  4. 先选中目录下的空白区域（表示选中test这个文件夹，会有蓝色的边框框中），然后新建一个.py文件，输入print('hello world!')
  5. 创建自动调试配置，此调试依赖于前面安装的Python Debugger插件，选中 Python Debugger，然后再选择Python文件即可，会自动打开生成的json文件

  6. 切换回py文件，按F5，VSCode的终端就会运行py文件中的代码，输入print里面的内容

至此，基础搭建就完成啦

其他补充点

  1. 修改调试的输出位置：如果觉得调试的内容在终端中输出，内容很杂乱，可以修改配置在调试控制台输出，可以修改以下文件配置，输出效果如下图（简洁很多）

  2. 在VSCode中切换不同的Python版本：在选择py文件的情况下，底部状态栏会显示当前对应的Python版本，点击版本号则可进行切换

  3. 生成虚拟环境：选择对应的Python版本之后，在终端输入 python -m venv .venv，就会在项目根目录下生成对应的.venv环境文件夹，可用来安装一些第三方包啥的

其他辅助插件推荐

  • autoDocstring - Python Docstring Generator：函数注释生成器
