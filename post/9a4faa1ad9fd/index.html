github sync github仓库：
https://github.com/kevinmkchin/Obsidian-GitHub-Sync

插件下载

  • 打开配置 -> 第三方插件（关闭安全模式）
  • 浏览 【社区插件市场】
  • 搜索 github sync

启用插件

右侧栏出现这个小图标，则表示启用成功

配置说明

  • GitHub username（GitHub用户名，填了token，可不填）
  • GitHub personal access token（个人访问令牌，必填）
  • GitHub repo URL for this vault（存放obsidian文件的仓库，格式：github.com/用户名/仓库名）
  • git binary location（电脑git的安装目录，需要注意路径的斜杠，“\” 要改成 “/”）
  • Auto sync at interval (自动更新的时间)

    配置完成后，重启 obsidian
